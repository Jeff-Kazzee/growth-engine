# Portable Prompts

These are platform-agnostic versions of the Growth Engine skills. No plugin system required — paste them into any capable AI assistant: Zo Computer, Claude, ChatGPT, Gemini, or anything else that can do web research and write files.

| Prompt | Use it to |
|---|---|
| `growth-profile-setup-prompt.md` | Build your profile + memory folder (run first) |
| `opportunity-radar-prompt.md` | Find evidence-backed emerging opportunities |
| `niche-research-wiki-prompt.md` | Turn a niche into a research knowledge base |
| `personal-brand-positioning-prompt.md` | Build truthful, memorable positioning |
| `growth-review-prompt.md` | Accountability check-in on your commitments |
| `growth-retro-prompt.md` | Improve your own playbook based on what worked |

## How to get the "memory" without a plugin

The plugin version persists state automatically. To get the same effect manually:

1. Run the profile setup prompt once; save the output as `growth-engine/profile.md` wherever you keep notes.
2. Whenever you run any other prompt, paste in (or attach) your `profile.md` and `playbook.md`.
3. Save what each run produces back into the folder (`decision-log.md`, `watchlist.md`, `proof-assets.md`).
4. On Zo Computer: keep the `growth-engine/` folder in your Zo files and just tell Zo to read it first — or install the skills from `../skills/` directly (same SKILL.md format).

The memory is the point. A brilliant one-shot answer you never revisit changes nothing; a mediocre plan you're held accountable to every two weeks changes everything.
